# Cha-GPT

A tiny static ChatGPT-style page that always replies with `Mawmawmaw`, `Mawmaw`, or a random-length `Maw` chain.

## Run locally

Open `index.html` in a browser, or serve the folder with any static file server.

## Host on GitHub Pages

1. Push this folder to a GitHub repository.
2. In the repository, open `Settings` -> `Pages`.
3. Set the source to the main branch and the root folder.
4. Save, then open the GitHub Pages URL after it finishes publishing.

No build step or API key is needed.
